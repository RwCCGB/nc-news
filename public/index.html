NC News API

Information about the APIs created for the NC News Project

Available Endpoints

Method Endpoint Description
GET/api/articlesList all articles
GET/api/articles/:article_idGet an article by ID
POST/api/articles/:article_id/commentsPost a new comment
PATCH/api/articles/:article_idUpdate article votes
DELETE/api/comments/:comment_idDelete a comment

GET /api/articles

Returns a list of all articles, sorted by date (default).

            {  
            "articles": [
                {
                "article_id": 1,
                "title": "Running with purpose",
                "topic": "coding",
                "author": "[username]",
                "body": "How to look like you mean to run somewhere.",
                "created_at": "2020-11-07T12:00:00.000Z",
                "votes": 10,
                "comment_count": 5
                },
                {
                "article_id": 2,
                "title": "My lovely article",
                "topic": "coding",
                "author": "[username]",
                "body": "An articles body",
                "created_at": "2020-11-08T12:00:00.000Z",
                "votes": 7,
                "comment_count": 2
                }
            ]
         }

GET /api/articles/:article_id

Returns a single article by its ID.

        {
            "article": {
                "article_id": 1,
                "title": "Eating Cheese: A Beginners Guide",
                "topic": "coding",
                "author": "[username]",
                "body": "I never knew I was eating cheese wrong!",
                "created_at": "2020-11-07T12:00:00.000Z",
                "votes": 10
            }
        }

POST /api/articles/:article_id/comments

Posts a new comment to the specified article.

Request Body:

            {
                "username": "[username]",
                "body": "I love this article!"
            }

Response:

        {
            "comment": {
                "comment_id": 42,
                "author": "[username]",
                "article_id": 1,
                "body": "I love this article!",
                "created_at": "2025-07-28T10:00:00.000Z",
                "votes": 0
            }
        }

PATCH /api/articles/:article_id

Updates the vote count of an article by the number provided.

Request Body:

            {
                "inc_votes": 1
            }

Response:

            {
                "article": {
                "article_id": 1,
                "votes": 11
                }
            }

DELETE /api/comments/:comment_id

Deletes the comment. Responds with status 204..
